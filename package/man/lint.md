#### Description

The `lint` command group provides tools for validating `.aux4` configuration files. It checks for structural correctness, naming conventions, reference integrity, and best practices.

Available subcommands:

- **run** — Execute the linter against `.aux4` files in a directory

#### Usage

```bash
aux4 lint <subcommand>
```

#### Example

```bash
aux4 lint run --dir ./my-package
```
